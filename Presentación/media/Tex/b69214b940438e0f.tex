Para cualesquiera dos clases de sucesiones de morfismos $[A\xrightarrow{x}B\xrightarrow{y}C]$ y $[A'\xrightarrow{x'}B'\xrightarrow{y'}C']$, denotamos $$[A\xrightarrow{x}B\xrightarrow{y}C]\bigoplus[A'\xrightarrow{x'}B'\xrightarrow{y'}C'] := \bigg[A\bigoplus A' \xrightarrow{x\bigoplus x'} B\bigoplus B' \xrightarrow{y\bigoplus y'} C\bigoplus C'\bigg].$$
